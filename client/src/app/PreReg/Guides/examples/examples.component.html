The Engineering group is also developing a few basic problem sets that are ideal for demonstration and tutorials.

When you first receive an account, you are allocated a set of tutorial 'projects'; these are live projects that you can observe and the modify as you wish to 'try out' how it works.

Fear not: you are allowed to reload the projects if you mess them up :-)

Here are some of the tutorial problems:

  • Find the 10th Root of 25 billion; this is like the square root only harder.
  • The Traveling Saleman Problem - who must visit each of a number of cities in the most efficient order.
