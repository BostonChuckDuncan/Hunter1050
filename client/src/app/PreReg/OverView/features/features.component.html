Features

Solution Hunter has developed new methods using the latest computing technologies that advance the state-of-the-art in this type of search engine.

  • Hunter applies a variety of "plug-ins" targeted for problem areas.
  • Hunter is targeted corporate use, but can be configured for small businesses.
  • It can run on Azure, AWS, or corporate Systems (Windows).
  • It is multi-threaded and can take full advantage of multiple local or remote computers for additional speed.
  • Hunter is highly configurable and can learn to improve its strategy to solve the current problem more efficiently. When it finds a better “hunter”, it will dynamically switch to the more powerful image without interruption.

Hunter itself is pursuing leading edge computing technologies to further improve its speed and assure stability well into the future.
